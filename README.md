# 🗒️ NotePad With BLoC

یک اپلیکیشن یادداشت ساده و کاربردی با استفاده از Flutter و الگوی مدیریت وضعیت BLoC ساخته شده است.

## ✨ ویژگی‌ها

- اضافه کردن یادداشت جدید
- ویرایش یادداشت‌های قبلی
- حذف یادداشت‌ها
- ذخیره‌سازی محلی یادداشت‌ها
- طراحی ساده و تمیز
- استفاده از معماری BLoC برای مدیریت وضعیت

## 🚀 تکنولوژی‌ها

- Flutter
- Dart
- BLoC
- Hive (برای ذخیره‌سازی محلی)
- Flutter UI Components
